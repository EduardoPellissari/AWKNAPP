# Como instalar no iPhone

1. Abra o link do app no Safari.
2. Toque no botao de compartilhar.
3. Escolha "Adicionar a Tela de Inicio".
4. Confirme o nome "awkn!".
5. Abra pelo icone criado na tela inicial.
